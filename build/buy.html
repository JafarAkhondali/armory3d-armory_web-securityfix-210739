Armory Preview Version

(current revision: 16.09)

Preview version will be avaialble soon! Before purchasing, please read the manual first to know what parts need more work.

Estimated price: USD 49 / EUR 43 / GBP 37

While we are still looking for ways to make Armory free, you can support the project directly and get the preview version. We do not plan to sell any separate modules or force splash screens. Everyone accessing the preview version will of course receive all future releases for free. Careful though, the preview version will likely have many rough edges and the experience may be frustrating!

Please note that eventually, as the project matures, we would love making it free. The funds are (unfortunatelly!) needed to actively develop project. No one would like to use an engine that is not well maintaned. We also plan to split part of the revenue to Blender and Kha, which are the basic blocks of Armory.

All revenue this project receives will be publicly disclosed on this page.
